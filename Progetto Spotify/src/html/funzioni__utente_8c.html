◆ isAutenticazione()

int isAutenticazione ( int  utenti_effettivi,
int *  posizione_utente 
)

Questa funzione permette di autenticare l'accesso dell'utente.
Per fare ciò, controlla prima se l'utente inserito in input è presente. Se l'utente è presente, egli potrà continuare con l'inserimento della password che avverrà
facendo apparire a schermo un "*" per ogni tasto premuto. Eventualmente l'utente avrà la possibilità anche di cancellare la password per un eventuale errore di battitura.

Postcondizione
Il ritorno sarà un intero
